# 🚀 Shreyas Srivastava — AI/ML Portfolio

A futuristic AI & Machine Learning themed personal portfolio website built to showcase projects, skills, certifications, and experience in Machine Learning, Generative AI, NLP, and LLM-powered applications.

Designed with a premium cyberpunk-inspired UI using neon green, electric blue, and deep black aesthetics combined with advanced animations and interactive user experiences.
